Back-End: Deliver Database to Front-End

This component deals with the parsing and delivery of the database from the back- to front- end.

Requirements

The Database Upload Form component will drop a file onto the server with the monkey database. This database is provided in a tab delimited file (see an example). Your responsibility is to parse this tab delimited file and return a JSON array of objects (e.g., JSON array of objects for the previous example). Furthermore, it may be a good idea to cache the JSON file you create. The cache would have to be recreated every time that a new database is uploaded, though.

You must use PHP for the server-side scripting language, because this project will eventually reside on the IWU Sun server which has PHP most readily available.

Coordination with Other Components

Be sure that the JSON output looks like the provided example.

Tools

Review the PHP documentation on reading CSV files. Also, note that the Sun server has PHP version 5.1.6, so you cannot use functions like str_getcsv.
